# Лабораторная работа №6: Обработка двумерных массивов и контейнеров

## Описание
Данная лабораторная работа посвящена изучению алгоритмов формирования, обработки и манипуляции двумерными массивами, а также работе с контейнерами стандартной библиотеки шаблонов (STL) языка C++.

**Университет:** НГТУ (Новосибирский государственный технический университет)

## Цели и задачи работы
Изучение алгоритмов формирования и обработки двумерных массивов, программирование и отладка программ формирования и обработки массивов (статических и динамических) и контейнеров STL.

## Задание к работе
Самостоятельно решить задачи в соответствии с индивидуальным вариантом на двух языках программирования (С++ и Go/Rust):

1. **Задание 1.** Обработка двумерных массивов.
2. **Задание 2.** Многоалфавитное шифрование/дешифрование с использованием алгоритма AES.
3. **Задание 3.** Методы решения систем линейных алгебраических уравнений.
4. **Задание 4.** Доска с монетами.
5. **Задание 5*.** Фракталы.

## Структура проекта
- `main.cpp` — основной файл с точкой входа и логикой тестирования.
- `*.h / *.cpp` — заголовочные файлы и модули реализации алгоритмов.
